# Alpha Vantage

Gives agents real-time and historical financial market data: stock quotes, forex rates, cryptocurrency prices, and 50+ technical indicators.

[Alpha Vantage API documentation](https://www.alphavantage.co/documentation/)

## What you need

| Field   | Required | Notes                        |
| ------- | -------- | ---------------------------- |
| API Key | Yes      | Secret — free or premium key |

## Getting your API key

1. Go to [alphavantage.co/support/#api-key](https://www.alphavantage.co/support/#api-key).
2. Fill in the claim form (free, no credit card).
3. Your key is displayed immediately — copy it into the credential form.

::: warning Free-tier limit
The free tier allows up to **25 requests per day**. If an agent makes frequent market queries, consider a premium key.
:::
