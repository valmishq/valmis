# Pinecone

Lets agents use the Pinecone vector database — semantic search and embedding storage. The API key is sent in a custom header.

[Pinecone API documentation](https://docs.pinecone.io/reference/api/introduction)

## What you need

| Field   | Required | Notes                            |
| ------- | -------- | -------------------------------- |
| API Key | Yes      | Secret — keys are project-scoped |

## Getting your API key

1. Sign in at [app.pinecone.io](https://app.pinecone.io).
2. Select the project the agent should work in — keys are scoped per project.
3. Open the **API Keys** section, create (or copy) a key, and paste it into the credential form.
