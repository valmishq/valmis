# Calendly

Lets agents read scheduling data, event types, and invitees from your Calendly account. Authenticates with a Personal Access Token sent as a Bearer token.

[Calendly API documentation](https://developer.calendly.com/api-docs/)

## What you need

| Field                 | Required | Notes  |
| --------------------- | -------- | ------ |
| Personal Access Token | Yes      | Secret |

## Getting your token

1. Sign in to Calendly and open [Integrations & apps → API & Webhooks](https://calendly.com/integrations/api_webhooks).
2. Click **Generate new token**.
3. Name it, generate, and copy the token into the credential form.
