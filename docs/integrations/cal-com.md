# Cal.com

Lets agents manage schedules, event types, bookings, and availability through the Cal.com API v2. The key is sent as a Bearer token.

[Cal.com API v2 documentation](https://cal.com/docs/api-reference/v2/introduction)

## What you need

| Field   | Required | Notes                                                            |
| ------- | -------- | ---------------------------------------------------------------- |
| API Key | Yes      | Secret — test keys start with `cal_`, live keys with `cal_live_` |

## Getting your API key

1. Sign in to Cal.com and go to **Settings → Security → API Keys**.
2. Create a new key (choose an expiry, or never).
3. Copy it into the credential form.
