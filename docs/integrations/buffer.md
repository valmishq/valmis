# Buffer

Lets agents schedule and manage social media posts through Buffer. The API key acts on behalf of your account and grants access to all organizations and channels in it.

## What you need

| Field   | Required | Notes  |
| ------- | -------- | ------ |
| API Key | Yes      | Secret |

## Getting your API key

1. Sign in to Buffer and open [Settings → API](https://publish.buffer.com/settings/api).
2. Generate an API key (or copy your existing one).
3. Paste it into the credential form.

::: warning Account-wide access
A Buffer API key is not scopeable — it gives access to everything in your Buffer account. Attach it only to agents you trust with your social channels.
:::
