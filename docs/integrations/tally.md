# Tally

Lets agents access your Tally forms, submissions, and account data. Authenticates with a personal API key sent as a Bearer token.

[Tally API key documentation](https://tally.so/help/api-keys)

## What you need

| Field   | Required | Notes                           |
| ------- | -------- | ------------------------------- |
| API Key | Yes      | Secret — keys start with `tly-` |

## Getting your API key

1. Sign in to Tally and open **Settings → API keys**.
2. Generate a new key.
3. Copy it (it starts with `tly-`) into the credential form.
